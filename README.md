# KAN-PROSPECT
An AI-driven framework for large-scale prediction of pharmacological effects and adverse drug reactions of natural products. KAN-PROSPECT combines graph neural networks, Kolmogorov–Arnold Networks (KAN), and transfer learning to improve robustness, representation learning, and generalization under limited-data conditions.
